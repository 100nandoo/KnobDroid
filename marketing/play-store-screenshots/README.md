# KnobDroid Play Store Screenshots

Local Next.js generator for KnobDroid's Google Play marketing screenshots.

## Run

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Exports

- `Phone`: 1080x1920
- `7" Portrait`: 1200x1920
- `7" Landscape`: 1920x1200
- `10" Portrait`: 1600x2560
- `10" Landscape`: 2560x1600
- `Feature Graphic`: 1024x500

Use `Export Visible Set` for the active device or `Export Every Set` to download the full Play Store package.
